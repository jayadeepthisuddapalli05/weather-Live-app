# 🌦️ Weather Live App

A simple and responsive weather web application that allows users to search for any city and get real-time weather data, air quality index (AQI), and a 5-day forecast.

---

## 🚀 Features

* 🔍 Search weather by city name
* 🌡️ Real-time temperature display
* ☁️ Sky condition description
* 📅 Current date & time
* 🌬️ Additional metrics:

  * Humidity
  * Pressure
  * Wind Speed
  * Feels Like temperature
* 🌫️ Air Quality Index (AQI)
* 📊 5-day weather forecast
* 📱 Fully responsive design

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Custom styling + Glassmorphism UI)
* JavaScript (Vanilla + jQuery)
* Bootstrap 5
* OpenWeatherMap API

## ⚙️ How It Works

* The app fetches weather data using the OpenWeatherMap API.
* It dynamically updates UI elements like temperature, AQI, and forecast.
* User input triggers API calls via search button or Enter key.

---

## 🔑 API Used

* OpenWeatherMap API

  * Current Weather
  * Air Pollution (AQI)
  * 5-Day Forecast

## ▶️ How to Run Locally

1. Clone the repository

```bash
git clone https://github.com/your-username/weather-live-app.git
```

2. Open the project folder

```bash
cd weather-live-app
```

3. Open `index.html` in your browser
